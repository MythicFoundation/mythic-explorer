# Mythic Explorer

Block explorer frontend for the Mythic L2 network. Displays blocks, transactions, accounts, and network statistics in a modern UI.

## Features

- **Block Explorer**: Browse blocks, transactions, and accounts
- **Stats Dashboard**: Network TPS, slot height, validator count, supply data
- **Transaction Detail**: Full transaction inspection with instruction decoding
- **Account Viewer**: Token balances, transaction history, program data

## Tech Stack

- **Frontend**: Next.js 14, Tailwind CSS, TypeScript
- **API Backend**: Mythic Explorer API (`https://api.mythic.sh`)
- **Brand**: Network green (#39FF14)

## Live

- **Explorer**: `https://mythic.sh` (integrated into main site)
- **API**: `https://api.mythic.sh`

## Setup

```bash
cd website
npm install
npm run dev
```
